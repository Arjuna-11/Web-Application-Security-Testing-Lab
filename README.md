# Web-Application-Security-Testing-Lab
web app security with kali and virtcoin setup

Technologies: Kali Linux, SQLMap, Linux, Web Security, Networking, Virtualization

This project focused on performing security analysis and vulnerability testing within a controlled web application lab environment using Kali Linux and industry-standard security tools. The environment simulated common web application vulnerabilities and attack scenarios to better understand web security weaknesses, authentication handling, session management, and input validation issues. Security testing tools such as SQLMap and browser-based developer tools were used to analyze application behavior, inspect requests and responses, and identify potential vulnerabilities within a sandboxed educational environment. The project also involved Linux-based tooling, virtualized testing environments, and security-focused troubleshooting workflows to gain hands-on experience with web application security assessment concepts.
